---Brief News---

делаю т.з. в первый раз, поэтому не на английском языке


--Main:
Создать сайт в форме ленты разбитой на страницы с краткой "выжимкой" новостей и ссылками на источники

--Сервер
OS: Ubuntu 22.04.2 LTS
Programming language: PHP 8.1.2-1
Web server: Apache/2.4.52
Backend framework: Laravel 10.13.0
Backend framework libraries: UI, doctrine/dbal
Frontend framework: отсутствует

--База Данных:
СУБД: MYSQL
Имя базы данных: brief_news
Пользователь: brief_news_user
Пароль: стандартный (не отображен, т.к. т.з. будет доступен на GitHub)
Дополнительно: Soft delete для пользователей, статей и ссылок, создать первого root (admin привилегии) пользователя


--Роли пользователей и разграничение прав:
Admin: CRUD + смена статуса пользователей (бан), смена статуса статей (бан), + права writer
Writer: CRUD статей, + права reader
Reader: CRUD своей учетной записи, настройка сортировки статей по умолчанию, сортировка статей по категориям, читать статьи
Unregistered: создать учетную запись, сортировка статей категориям, читать статьи


--Интефейс общий:
Header: небольшой фрагмент страницы с пользовательским интерфейсом
Регистрация, Log in/Log out, ссылка на редактирование своего профиля.

Body новости:
Рубрики новастей:
Горизонтальная полоса из кнопок: All, Policy, Economy, Science, Technologies, Sport, Other, World, Local.
Распологается выше ленты и отображается только с ней.
При нажатии одной или нескольких кнопок в ленте отображаются новасти только выбранных рубрик.
Кнопка All сбрасывает остальные кнопки.

Новости:
Вертикальная лента с заголовками новостей по 10 шт на странице.
Последняя новость по дате создания в верху списка.
При клике на заголовок открывается тело новости с "выжимкой" новости. В нижней
части автор "пересказа" и ссылки на источники.

Body редактирование учетной записи:
автозаполняемая форма с данными пользователя и кнопками сохранить изменения и удалить текущего пользователя.
