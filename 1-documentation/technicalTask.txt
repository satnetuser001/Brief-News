--Main:
Создать сайт в форме ленты разбитой на страницы с краткой "выжимкой" новостей и ссылками на источники

ОС сервера: Ubuntu 22.04.2 LTS
Язык программирования: PHP 8.1.2-1
Backend framework: Laravel 10.13.0
Backend framework libraries: ...
Frontend framework: отсутствует


--Роли пользователей и разграничение прав:
Admin: CRUD + смена статуса пользователей (бан), CRUD + смена статуса статей (бан, невидимая - на доработке)
Writer: CRUD + смена статуса своих статей
Reader: CRUD своей учетной записи, сортировка статей категориям, читать статьи
Unregistered: создать учетную запись, сортировка статей категориям, читать статьи


--Интефейс Reader:
Header: небольшой фрагмент страницы с пользовательским интерфейсом
Регистрация, Log in/Log out, ссылка на редактирование своего профиля.

Body лента: в верхней части кнопки с названиями рубрик. При нажатии на кнопку происходит
новая выборка из базы данных с учетом выбранных рубрик.
Ниже лента с заголовками новостей разбитая на страницы по 10 шт.
Последняя новость по дате публикации в верху списка.
При клике на заголовок открывается тело новости с "выжимкой" новости. В нижней
части автор "пересказа" и ссылки на источники.

Body редактирование учетной записи: автозаполненая форма с данными пользователя
и кнопками сохранить изменения и удалить текущего пользователя.

Footer: отсутствует


--База Данных:
СУБД: MYSQL
База данных: brief_news
Пользователь: brief_news_user
Пароль: стандартный (не отображен, т.к. т.з. бедет доступно на GitHub)
Реализовано: Soft delete для пользователей и статей, создан первый root пользователь
